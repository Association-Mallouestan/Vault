# Nettoyer la cuisine extérieur 
[← Liste des tâches](../)

## Description
Cette tâche consiste ramener et ranger les éléments de l'auberge oubliés dehors type plats, épices etc. et nettoyer les supports de cuisine ainsi que les tables si besoin.  


## Points à faire

- [ ]  Tu auras besoin de : la brosse située dans le garde-manger à côté du robinet d'eau, elle aura l'inscription "brosse de table". D'une bascine d'eau que tu trouveras dans la salle de bain. 
- [ ] Il s'agit de nettoyer le support près des brûleurs ainsi que les tables. De ranger autour des brûleurs, si tu trouves par exemples des sacs cabas vides, les remettres dans le petit abris à bois. De ramener les plats ou les épices etc dans l'auberge et de les remettre à leur place.
